# Milestone Summary: v2.0 iOS App — Shortcut Dictation

**Completed:** 2026-04-22
**Status:** SHIPPED

## Milestone Overview
Transformed Dicticus into a multi-platform solution by introducing a native iOS application. v2.0 focuses on bringing high-accuracy, on-device transcription to iPhone and iPad with deep system integration via Siri Shortcuts and Action Button support.

## Key Accomplishments

### 1. Shared Core Pipeline
- Extracted and unified transcription logic into a cross-platform `Shared/` module.
- Integrated `FluidAudio` for iOS arm64 with support for Parakeet TDT v3 models.
- Ported Inverse Text Normalization (ITN) and Custom Dictionary logic to work seamlessly on both platforms.

### 2. High-Accuracy iOS Dictation
- Implemented `IOSTranscriptionService` with `AVAudioSession` management.
- Achieved ~2.7GB model provisioning with a graceful background warmup and download UI.
- Integrated `NaturalLanguage` for auto-detection of English and German.

### 3. System Integration (iOS)
- **App Intents:** Created the `Start Dictation` shortcut for Siri and Shortcuts app activation.
- **Live Activity:** Developed a Dynamic Island and Lock Screen experience for real-time recording feedback.
- **Shortcuts:** Provided pre-defined Siri phrases for instant voice activation.

### 4. Local Persistence & UI
- **History:** Integrated `GRDB` for a local SQLite transcription database with FTS5 search.
- **Universal Layout:** Adaptive UI supporting iPhone (Tabs) and iPad (Sidebar).
- **Settings:** Comprehensive management for dictionary entries and model state.

## Requirement Fulfillment
- **Total Requirements:** 22/22 (100% met)
- **Core Value Check:** Fulfills the "Press a key, speak, release" promise on mobile while maintaining 100% privacy.

## Next Steps
- **v2.1 Planning:** Focus on iCloud Sync (CloudKit) to bridge the gap between macOS and iOS history/dictionary.
- **Keyboard Extension:** Researching memory-efficient "Quick Dictation" for direct text field entry on iOS.
