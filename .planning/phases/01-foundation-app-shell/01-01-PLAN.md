**DicticusApp.swift (per D-06 — SF Symbol template, per APP-01 — no main window):**
```swift
import SwiftUI

@main
struct DicticusApp: App {
    var body: some Scene {
        MenuBarExtra {
            MenuBarView()
        } label: {
            Image(systemName: "mic")
        }
        .menuBarExtraStyle(.window)
    }
}
```

Key points:
- NO `WindowGroup` scene — menu bar only (APP-01)
- SF Symbol `"mic"` for the menu bar icon — monochrome template image that auto-adapts to light/dark mode (per D-06)
- `.menuBarExtraStyle(.window)` — required for custom UI (ProgressView, badges) in later plans; `.menu` style only supports Button/Toggle/Divider

**MenuBarView.swift (per D-05 — minimal Phase 1 dropdown):**
```swift
import SwiftUI

struct MenuBarView: View {
```

**Info.plist — configure as a menu bar agent app:**
```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>LSUIElement</key>
    <true/>
    <key>NSMicrophoneUsageDescription</key>
    <string>Dicticus needs microphone access to transcribe your speech. Your audio never leaves this device.</string>
</dict>
</plist>
```

- `LSUIElement = true` hides the app from the Dock (menu bar only per APP-01)
- `NSMicrophoneUsageDescription` is mandatory for microphone permission prompt (APP-02, used in Plan 02)

**Dicticus.entitlements (per INFRA-05 — unsandboxed distribution):**
```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>com.apple.security.app-sandbox</key>
    <false/>
    <key>com.apple.security.device.audio-input</key>

```

- Sandbox explicitly disabled (INFRA-05 — required for system-wide hotkeys in Phase 3)
- Audio input entitlement enabled (required for microphone access)
- Hardened Runtime is enabled at the Xcode project level (build setting `ENABLE_HARDENED_RUNTIME = YES`), not in the entitlements file
