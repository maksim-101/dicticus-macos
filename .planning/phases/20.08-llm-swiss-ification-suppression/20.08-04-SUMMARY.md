## Trade-offs absorbed by the variant (g15) verbatim contract

The variant (g15) prompt ships verbatim per the canonical brief.
Per-feature consequences for the German path:

- Custom `INSTRUCTION` override (UserDefaults-backed) no longer applies.
- `DICTIONARY:` context (user-defined replacements) no longer applies.
- Phase 20.06 STRICT speaker-explicit currency anchor (`never substitute
  Franken with Euro`) no longer emitted. Currency preservation now
  rests on the third few-shot example + Plan 02's `gateLLMDialect`
  backstop + upstream `CurrencyAntiFlip` detection.

These trade-offs were known at commit time. The currency anchor
removal is now implicated in the UAT failure (see below).
