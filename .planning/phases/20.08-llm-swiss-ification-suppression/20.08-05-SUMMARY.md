**1. [Rule 1 — Bug] Plan splice-2 KORRIGIERT-line wording corrected for grammatical capitalisation**

- **Issue:** Plan `<action>` Splice 2 specified KORRIGIERT line as `Ich habe bestimmt über 99 Franken ausgegeben, vielleicht sogar um die 102.50 Franken.` — i.e. `vielleicht` lowercase mid-sentence. But Plan `<behavior>` Behaviour 3 + Task B Test 3 + Task A acceptance grep all required `Vielleicht sogar um die 102.50 Franken` (capitalised V). The ORIGINAL line is unchanged from Splice 2.

- **Commit:** 2cab498
