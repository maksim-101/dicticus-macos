<objective>
Patch `Shared/Models/CleanupPrompt.swift::buildGermanVariantG15` to close R-G15-01 (currency-digit truncation: `102.50 Franken` → `12.50 Franken`, reproduced cross-platform on macOS Release + iOS Release in the 2026-05-01 UAT). The fix is additive and dual-layer: (1) one short German directive sentence appended to the INSTRUCTION block forbidding digit/amount mutation, AND (2) one new 5th ORIGINAL/KORRIGIERT exemplar demonstrating preservation of a 3-digit decimal currency in colloquial-modal context (`vielleicht sogar um die 102.50 Franken`) — the exact contextual shape variant (g15) over-edited.

**Hybrid (rule + exemplar) chosen over Option 1 (rule-only) or Option 2 (exemplar-only).** Rationale per VARIANT-G-RATIONALE §10.2 + Wave-B finding ("few-shot is stabilising when examples each demonstrate a distinct rule on diverse content"): a rule statement gives Gemma 4 E2B explicit coverage of the entire number/amount class (not just the specific exemplar shape), while a same-shape exemplar provides the in-context-learning signal that Wave-B established as the strongest stabiliser at temp 0.1. The combined budget is ~25 tokens (sentence) + ~30 tokens (exemplar pair) ≈ 55 tokens — well below the g14 disqualification threshold, which was a 4-pair verbose umlaut anchor (~200 tokens, F2 prompt-leak observed). Each new artefact demonstrates a *distinct* rule (the existing 4 exemplars cover anglicism preservation, broken-grammar reconstruction, 4-digit currency preservation, and dialect-anti-drift; the new 5th covers 3-digit decimal currency preservation under contextual implausibility — a non-overlapping rule).

The primary anti-Swiss-ification contract held cleanly on both platforms in the 2026-05-01 UAT (zero Swiss particles, zero `ß`, Standard High German). This plan does NOT revert variant (g15); it patches the digit-preservation gap additively. The two-layer German conditional (D3), the four existing exemplars, RULE 1, the anti-dialect directive, and the anglicism two-tier rule remain byte-identical. The non-German path is untouched.

Out of scope: R-G15-02 (iOS-only language-detection bleed into preamble) — flagged in the threat register as a known upstream-of-prompt issue requiring a separate phase (Parakeet/FluidAudio per-utterance language-detection). os_log capture of `gateLLMDialect` firings during UAT is flagged as nice-to-have but not gating.

Output: Patched `buildGermanVariantG15`, four new R6 contract tests, and a `20.08-05-UAT-RESULTS.md` artifact recording cross-platform digit-exact preservation of `102.50 Franken`.

```swift
private static func buildGermanVariantG15(text: String, swissOrthography: Bool) -> String {
    let orthographyClause = swissOrthography
        ? " mit Schweizer Rechtschreibung (ss statt ß, Umlaute ä/ö/ü bleiben)"
        : ""
    let sanitizedText = sanitizeControlTokens(text)

    var prompt = "<start_of_turn>user\n"
    prompt += "Bereinige die folgende deutsche Sprachaufnahme. "
    prompt += "Schreibe Standard-Hochdeutsch\(orthographyClause). "
    prompt += "Verwende KEINEN Schweizerdeutsch-Dialekt — schreibe \"Woche\" nicht \"Wuche\", \"Zürich\" nicht \"Züri\", \"ich gehe\" nicht \"i gang\". "
    prompt += "Etablierte englische Fachbegriffe bleiben Englisch (Deadline, Meeting, Workaround, E-Mail, Team, Product Owner, Release). "
    prompt += "Untypische englische Adjektive oder Verben in deutschen Sätzen ins Deutsche übertragen — \"realistic\" → \"realistisch\", \"awesome\" → \"toll\", \"appreciate\" → \"schätzen\". "
    prompt += "Gib genau eine bereinigte Version aus, sonst nichts.\n"
    prompt += "\n"
    prompt += "ORIGINAL: ich habe heute mit dem product owner gesprochen über die deadline und er meinte das ist nicht realistic.\n"
    prompt += "KORRIGIERT: Ich habe heute mit dem Product Owner über die Deadline gesprochen, und er meinte, das ist nicht realistisch.\n"
    prompt += "\n"
    prompt += "ORIGINAL: ich gestern gehen markt und kaufen viele apfel weil ich brauchen für kuchen.\n"
    prompt += "KORRIGIERT: Ich bin gestern auf den Markt gegangen und habe viele Äpfel gekauft, weil ich sie für den Kuchen brauche.\n"
    prompt += "\n"
    prompt += "ORIGINAL: das hotel hat ungefähr 1250 franken 20 gekostet und das war zu viel.\n"
    prompt += "KORRIGIERT: Das Hotel hat ungefähr 1250 Franken 20 gekostet, und das war zu viel.\n"
    prompt += "\n"
    prompt += "ORIGINAL: letzte woche war ich in zürich auf einer grossen konferenz.\n"
    prompt += "KORRIGIERT: Letzte Woche war ich in Zürich auf einer grossen Konferenz.\n"
    prompt += "\n"
    prompt += "ORIGINAL: \(sanitizedText)\n"
    prompt += "KORRIGIERT:<end_of_turn>\n"
    prompt += "<start_of_turn>model\n"

    return prompt
}
```
