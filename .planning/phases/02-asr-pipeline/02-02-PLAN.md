## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HuggingFace Hub -> ModelWarmupService | Model files downloaded from remote on first launch (D-09) |
| ModelWarmupService -> TranscriptionService | WhisperKit instance handoff at warmup completion |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-05 | Tampering | ModelWarmupService model download | accept | WhisperKit downloads from official argmaxinc/whisperkit-coreml HuggingFace repo. Model integrity depends on HuggingFace infrastructure. Risk is low for a local dictation app with no sensitive model fine-tuning. |
| T-02-06 | Denial of Service | ModelWarmupService.warmup | mitigate | Guard in warmup() prevents duplicate initialization calls. Error state communicated to user via "Model load failed. Restart app." message. No retry loop that could spin. |
| T-02-07 | Information Disclosure | Model cache on disk | accept | WhisperKit caches CoreML models in standard cache directory. Single-user desktop app -- file permissions follow macOS user model. No additional encryption needed for ML model weights. |
</threat_model>
