# Phase 19.7 Plan 01: install/uninstall scripts + README hygiene Summary

Shipped dev-only `scripts/install-local.sh` (98 lines / 3325 bytes) and universal `scripts/uninstall.sh` (80 lines / 2390 bytes) that resolve UAT row M2 multi-install TCC pollution; added 13-line "Manual uninstall" README section. `scripts/build-dmg.sh` is unchanged (D-02 honored).

## What Shipped

### `scripts/install-local.sh` — 98 lines, 3325 bytes

Dev-only consolidator. Six numbered steps:

1. Verify build artefact at `macOS/build/Build/Products/Release/Dicticus.app`.
2. Quit running Dicticus — `osascript -e 'tell application id "com.dicticus.macos" to quit'` then `pgrep -x Dicticus` retry loop (1s sleep × 3) then `pkill -f Dicticus.app` fallback.
3. Enumerate stale copies via the verbatim `mdfind` predicate.
4. `mv` each stale copy to `~/.Trash/Dicticus-stale-<TIMESTAMP>-<parent>.app` — never `rm -rf`.
5. Move existing `/Applications/Dicticus.app` to `~/.Trash/Dicticus-prev-<TIMESTAMP>.app`, then `cp -R` the fresh build into place.
6. `open -a Dicticus`, then echo a TCC reminder pointing at System Settings > Privacy & Security.

**Verbatim mdfind predicate shipped:**

```
mdfind "kMDItemCFBundleIdentifier == 'com.dicticus.macos'"
```

**Five exclusion guards shipped (Step 3):**

| Guard | Pattern |
|-------|---------|
| Whitelist | `*/Dicticus.app` (anything else continues) |
| Canonical | `"$CANONICAL_APP"` (= `/Applications/Dicticus.app`) |
| Trash | `"$HOME/.Trash"/*` |
| Time Machine local | `*/.MobileBackups/*` and `/Volumes/com.apple.TimeMachine.localsnapshots/*` |
| Time Machine network | `*/Backups.backupdb/*` |

**No `rm -rf` in install-local.sh** — verified by `grep -qE "rm -rf" scripts/install-local.sh` returning no match (Trash-only by design, T-19.7-01 mitigation). Stale copies and the previous canonical install are both `mv`'d to ~/.Trash with timestamped names.

### `scripts/uninstall.sh` — 80 lines, 2390 bytes

Universal uninstaller. Five steps:

1. Quit running instance (same osascript+pgrep+pkill shape as install-local.sh).
2. `rm -rf /Applications/Dicticus.app` (silent if absent).
3. Glob-remove `~/Library/Preferences/com.dicticus.*.plist`; `launchctl unload` + remove `~/Library/LaunchAgents/com.dicticus.app.plist`.
4. **Interactive prompt** for `~/Library/Application Support/Dicticus` (~3 GB) — `read -r -p "  Delete this directory? [y/N] " reply`, defaults to NO.
5. Echo TCC info block: System Settings > Privacy & Security > Microphone / Accessibility / Input Monitoring.

T-19.7-04 mitigation grep-verified — only `[yY]` / `[yY][eE][sS]` proceed; everything else keeps the directory.
