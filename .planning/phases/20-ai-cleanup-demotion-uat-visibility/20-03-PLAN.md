FillerWordRemover (this plan creates):
```swift
public enum FillerWordRemover {
    public static let germanFillers: Set<String>   // {"äh","ähm","ehm","hmm"}
    public static let englishFillers: Set<String>  // {"uh","um","umm","er","erm"}

}
```
Behaviour: case-insensitive word-boundary regex match per language; consume one trailing
`,?\s*` so orphan commas don't survive ("Das ist, äh, gut" → "Das ist gut"). Re-capitalize the
next word when a sentence-initial filler is removed.

SelfCorrectionResolver (this plan creates):
```swift
public enum SelfCorrectionResolver {
    public static func resolve(_ text: String, language: String) -> String
}
```
Behaviour: only fires when connector is preceded by a comma (precision over recall). Connectors:
de: `ich meine`, `besser gesagt`, `genauer gesagt`, `oder vielmehr`, `oder besser`
en: `I mean`, `I meant`, `or rather`, `or better`, `scratch that`
Backward window ≤ 3 tokens. If no clear noun-shaped reparandum within window → abort.

SwissNumberFormatter.foldCurrencyUnits (this plan adds):
```swift
extension SwissNumberFormatter {
    public static func foldCurrencyUnits(_ text: String) -> String
}
```
Patterns (zero-pad cents to 2 digits):
  - `(\d+)\s+(Franken|CHF)\s+(\d{1,2})\s+(Rappen|Rp\.?)` → `CHF \1.\3`
  - `(\d+)\s+(Euro|EUR|€)\s+(\d{1,2})\s+(Cent|Ct\.?)` → `€\1.\3`
  - `(\d+)\s+(Dollar|USD|\$)\s+(\d{1,2})\s+(Cent[s]?)` → `USD \1.\3`
  - `(\d+)\s+(Pfund|Pounds?|GBP|£)\s+(\d{1,2})\s+(Pence|p\.?)` → `GBP \1.\3`
Runs BEFORE bridgeCrossTokenDecimal in `format(_:)` — so once collapsed, the lower-precision
bridge becomes a no-op. Idempotent on already-folded input (negative-lookbehind unnecessary
since the pattern itself does not match the folded form).

    /// Phase 20 D-02: deterministic rules-first cleanup. Runs before the optional LLM stage in
    /// TextProcessingService. Composition is fixed and order-sensitive: filler removal must precede
    /// self-correction (so a filler embedded in a connector phrase doesn't break detection); currency

    public final class RulesCleanupService {
