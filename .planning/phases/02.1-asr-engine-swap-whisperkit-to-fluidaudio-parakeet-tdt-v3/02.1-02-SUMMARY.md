# Phase 2.1 Plan 02: Rewrite TranscriptionService for FluidAudio Summary

FluidAudio-based TranscriptionService with AVAudioEngine recording, AVAudioConverter resampling, Silero VadManager VAD, and NLLanguageRecognizer post-hoc language detection — zero WhisperKit references remain anywhere in the Dicticus source tree.

## What Was Built

- `TranscriptionService.swift`: Fully rewritten — `import WhisperKit` replaced with `import FluidAudio`, `import AVFoundation`, `import NaturalLanguage`. Private properties changed from `whisperKit: WhisperKit` to `asrManager: AsrManager`, `vadManager: VadManager`, `audioEngine: AVAudioEngine`, `audioSamples: [Float]`, `sampleRate: Double`. `init(whisperKit:)` replaced with `init(asrManager:vadManager:)`. `startRecording()` now uses AVAudioEngine input tap. `stopRecordingAndTranscribe()` resamples via AVAudioConverter (primary) or linear interpolation (fallback), then runs Silero VAD via `vadManager.process()`, then `asrManager.transcribe()`, then `detectLanguage()` via NLLanguageRecognizer. `silenceThreshold` default changed from 0.3 to 0.5. `nonisolated(unsafe)` removed. DEBUG helpers: `isFluidAudioAvailable()`, `makeForTesting()`, `testDetectLanguage()`, `testRestrictLanguage()`. Public API preserved verbatim (D-04).
- `DicticusApp.swift`: `import WhisperKit` replaced with `import FluidAudio`. `onChange` handler now reads `warmupService.asrManagerInstance` and `warmupService.vadManagerInstance` and passes them to `TranscriptionService(asrManager:vadManager:)`.
- `TranscriptionResult.swift`: Doc comments cleaned — all WhisperKit-specific language removed; struct itself unchanged (D-05).
- `TranscriptionServiceTests.swift`: 17 test methods total. `import FluidAudio` added. File header updated. `testTranscriptionErrorCasesExist` now checks all 6 error cases (was 4). Three new `testDetectLanguage*` tests added (NLLanguageRecognizer, no model needed). `testVadManagerIntegration` added (TRNS-04 coverage). `makeServiceOrSkip()` uses `isFluidAudioAvailable()` and `makeForTesting()`. `testDefaultSilenceThreshold` expects 0.5. All 59 tests pass.

## Decisions Made

| Decision | Rationale |
|----------|-----------|
| AVAudioConverter as primary resampler | Apple's high-quality resampler, no external dependency, recommended in RESEARCH.md |
| Linear interpolation as fallback | Simple, zero-dependency safety net if AVAudioConverter format setup fails |
| silenceThreshold = 0.5 | Silero VAD outputs probability 0.0–1.0; 0.5 is the natural midpoint per RESEARCH.md recommendation |
| nonisolated(unsafe) removed | AsrManager is a proper Swift actor; cross-actor await works naturally without the unsafe workaround |
| NLLanguageRecognizer constrained to [.german, .english] | Parakeet TDT v3 outputs no language code; Apple's framework provides reliable on-device language ID for de/en |
