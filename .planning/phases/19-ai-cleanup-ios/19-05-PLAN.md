## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-19-05-01 | T (Tampering) | AppGroup UserDefaults values | accept | AppGroup is sandboxed to app identity; only extensions signed with the same team ID can read/write. No PII in these keys. |
| T-19-05-02 | I (Info Disclosure) | Download URL visible in app | accept | URL is public (ungated HuggingFace link documented in CLAUDE.md and RESEARCH.md). No risk. |
| T-19-05-03 | D (Denial of Service) | User leaves Settings during download | mitigate | Downloader `@StateObject` deallocates when Settings dismisses ‚Äî in-flight task is cancelled.
