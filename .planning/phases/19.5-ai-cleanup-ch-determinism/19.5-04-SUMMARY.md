## W8 Closeout

W8 lock (raw `text` vs `sanitizedText` in detectCurrencies) honored. The call site is:

```swift
let detectedCurrencies = CurrencyAntiFlip.detectCurrencies(in: text)
```

The parameter is the raw `text` function argument, not `sanitizedText`. Rationale: `sanitizeControlTokens` only strips Gemma turn tokens (`<start_of_turn>`, `<end_of_turn>`, `<bos>`, `<eos>`) which never appear in user dictation, so detection on raw text is correct AND avoids requiring code reordering of the existing `sanitizedText` line. Acceptance grep `CurrencyAntiFlip.detectCurrencies(in: text)` confirms this.
